# 功能展示 / Features Showcase

## 系统概览 / System Overview

本网络性能监测工具提供了全面的网络监控、设备管理和性能分析功能。

## 主要功能模块 / Main Features

### 1. 用户认证 / User Authentication

#### 登录界面 / Login Page
- 现代化的登录界面设计
- 安全的用户认证机制
- 记住登录状态
- 表单验证

**功能特点**:
- ✅ 用户名/密码登录
- ✅ 输入验证
- ✅ 错误提示
- ✅ 自动跳转到仪表板

#### 注册界面 / Register Page
- 友好的注册表单
- 实时表单验证
- 密码强度检查
- 邮箱格式验证

**功能特点**:
- ✅ 用户名唯一性检查
- ✅ 邮箱格式验证
- ✅ 密码确认匹配
- ✅ 注册成功自动登录

### 2. 仪表板 / Dashboard

#### 系统概览
仪表板提供了系统的整体概览，包括：

**实时统计卡片**:
- 📊 设备数量统计
- 📈 上传速度实时显示
- 📉 下载速度实时显示
- ⚠️ 活动警报数量

**系统状态监控**:
- 💻 CPU使用率 (进度条可视化)
- 🧠 内存使用率 (颜色警示)
- 💾 磁盘使用率 (百分比显示)

**最近警报**:
- 📋 时间轴形式展示
- 🔴 根据严重程度着色
- 📅 时间戳显示
- 💬 详细警报消息

**界面特点**:
- 清晰的布局设计
- 实时数据更新（30秒自动刷新）
- 响应式卡片布局
- 直观的图标系统

### 3. 设备管理 / Device Management

#### 设备列表
显示所有注册的网络设备：

**表格字段**:
- 设备名称
- IP地址
- 设备类型（路由器、交换机、服务器等）
- 状态（在线/离线）
- 最后在线时间

**操作功能**:
- ➕ 添加新设备
- ✏️ 编辑设备信息
- 🗑️ 删除设备
- 🔍 设备状态监控

#### 添加/编辑设备对话框
- 📝 设备名称输入
- 🌐 IP地址验证
- 🔧 设备类型选择
  - 路由器 (Router)
  - 交换机 (Switch)
  - 服务器 (Server)
  - 电脑 (Computer)
  - 其他 (Other)

**验证规则**:
- IP地址格式验证
- 必填字段检查
- 实时表单验证

### 4. 实时监控 / Real-time Monitoring

#### 网络流量监控
实时显示网络流量统计：

**上传流量卡片**:
- 📤 总上传字节数
- 📦 发送数据包数量
- 格式化显示（B/KB/MB/GB）

**下载流量卡片**:
- 📥 总下载字节数
- 📦 接收数据包数量
- 自动单位转换

**刷新功能**:
- 🔄 手动刷新按钮
- ⏱️ 自动刷新（5秒间隔）
- 实时数据更新

#### 网速测试
- 🚀 一键测速功能
- 📊 下载速度显示（Mbps）
- 📊 上传速度显示（Mbps）
- ⏳ 测速进度指示

#### 系统资源监控
显示系统硬件资源使用情况：

**监控指标**:
- CPU核心数
- 总内存容量
- 可用内存
- 磁盘总空间
- 磁盘剩余空间

**显示特点**:
- 清晰的标签-数值对
- 自动格式化
- 易于理解的布局

#### 历史数据表格
查看历史网络流量记录：

**时间范围选择**:
- 最近1小时
- 最近6小时
- 最近24小时
- 最近7天

**数据列**:
- 时间戳
- 上传字节
- 下载字节
- 发送数据包
- 接收数据包

**功能**:
- 分页显示
- 最大显示400条
- 时间格式化
- 数据排序

### 5. 数据包捕获与分析 / Packet Capture

#### 抓包配置
灵活的抓包参数设置：

**协议选择**:
- 🌐 所有协议 (All)
- 🔵 TCP协议
- 🟢 UDP协议
- 🟡 ICMP协议
- ⚪ IP协议

**参数配置**:
- 📊 数据包数量（10-1000）
- ⏱️ 超时时间（5-60秒）
- 🎯 精确控制

**操作按钮**:
- ▶️ 开始抓包
- 📊 实时进度
- ✅ 完成通知

#### 数据包列表
详细的数据包信息展示：

**表格字段**:
- 🕐 捕获时间
- 🏷️ 协议类型（彩色标签）
- 📍 源IP地址
- 📍 目标IP地址
- 🔌 源端口
- 🔌 目标端口
- 📏 数据包长度
- ℹ️ 附加信息

**过滤功能**:
- 协议类型过滤
- 刷新数据
- 实时更新

**标签颜色**:
- TCP: 蓝色
- UDP: 绿色
- ICMP: 橙色
- IP: 灰色

#### 协议统计
可视化的协议分析：

**统计卡片**:
- 协议名称
- 数据包数量
- 总字节数
- 百分比占比

**展示方式**:
- 网格布局
- 彩色卡片
- 清晰的数据展示

### 6. 统计分析 / Analytics

#### 网络负载分析
全面的网络统计信息：

**摘要统计**:
- 📤 总上传流量
- 📥 总下载流量
- 📦 总发送数据包
- 📦 总接收数据包

**时间范围**:
- 1小时
- 6小时
- 24小时
- 7天

#### 警报管理
智能警报系统：

**活动警报标签页**:
- 警报类型
- 严重程度（标签显示）
- 详细消息
- 创建时间
- 解决按钮

**已解决警报标签页**:
- 历史警报记录
- 创建和解决时间
- 警报详情

**严重程度标签**:
- 🔵 信息 (Info)
- 🟡 警告 (Warning)
- 🔴 错误 (Error)
- 🔴 严重 (Critical)

#### 阈值配置
可自定义的警报阈值：

**配置项**:
- 🖥️ CPU阈值（50-100%）
- 🧠 内存阈值（50-100%）
- 💾 磁盘阈值（50-100%）

**控制方式**:
- 滑块调节
- 数值输入
- 实时预览

**保存功能**:
- 💾 保存到本地存储
- ✅ 保存成功提示
- 🔄 持久化配置

#### 系统状态卡片
实时系统状态监控：

**进度条显示**:
- CPU使用率
- 内存使用率
- 磁盘使用率

**颜色编码**:
- 🟢 绿色: 正常（< 阈值的80%）
- 🟡 黄色: 警告（阈值的80%-100%）
- 🔴 红色: 超限（> 阈值）
